15.1.0 Creating, editing and deleting whole {{eval}}s

To set up or edit an {{eval}} template click 'Admin -> system administration' to go to the admin menu, then 'Add, edit and remove {{eval}} templates'. You need to be a system administrator.

To start using {{eval}}s, you will need to set up one or more by entering the questions you wish to answer. To do this you will need administrator rights: click on 'Admin' on the main menu. From the admin menu click 'Add, edit and remove {{eval}} templates' in the 'Manage {{eval}}s' box. Section 18.5.0 Setting up {{eval}}s explains how to do this in detail.
