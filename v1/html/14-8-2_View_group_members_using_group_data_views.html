14.8.2 View {{group}} members using {{group}} data views

To view more information about the members of a {{group}}, you can set up '{{group}} data views' (see section 14.8.0 {{Group}} data views ) . To use a {{group}} data view, right-click on the {{group}}, hover over 'view full details' and choose the data view to use.
