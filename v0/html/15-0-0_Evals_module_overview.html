15.0.0 {{Eval}}s module overview

The {{Eval}}s Module enable you to store and analyse survey-type data. With the {{eval}} direct module you can also collect this information online (see 23.0.0 {{Eval}}s Direct Module).

The {{Eval}}s Module enable you to store and analyse survey-type data. You can set up as many surveys as needed, with as many questions of different types in them. You can then analyse the results in various ways - see section 17.5.0 {{Eval}} {{report}} for more details.

{{Eval}}s can be used as another way of tracking impact on your {{user}}s - see section 3.4.0 {{Outcome}}s for a discussion of the difference between {{outcome}}s and {{eval}}s in {{Lamplight}}.
