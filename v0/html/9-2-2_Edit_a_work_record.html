9.2.2 Edit a {{work}} record

To edit a record, right-click on a {{work}} record and click 'edit'. You can also click the edit button when viewing a record.

To edit a {{work}} record, click 'edit' on the context menu. A popup window will appear which will allow you to edit the record (see section 9.1.0 Adding a {{work}} record for more on adding {{work}} records)
