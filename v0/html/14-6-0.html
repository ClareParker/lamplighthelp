14.6.0 Editing {{group}}s

To edit a {{group}}, click {{Group}}s -> View all in the main menu. Right- click on the {{group}} you wish to edit and select 'edit' from the context menu.

To edit a {{group}}, click {{Group}}s -> View all in the main menu. Right- click on the {{group}} you wish to edit and select 'edit' from the context menu.

A window will open with the same screen you see when adding a {{group}} (see section 14.1.0 Creating an {{auto group}} or section 14.2.0 Creating a {{manual group}}, depending on whether you are editing an {{auto group}} or {{manual group}}). Make any changes you need, and click the 'Save' button in the bottom-right of the popup window.
